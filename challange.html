بخش اول: ساختن آرایه با متد Array
{
  "id": 1,
  "name": "Alireza",
  "email": "alireza@example.com",
  "age": 27,
  "city": "Tehran",
  "isActive": true,
  "role": "admin"
}

با استفاده از متد Array یک آرایه ۱۰ عضوی از آبجکت بالا بسازید.
مقدار id در هر عضو از ۱ تا ۱۰ به ترتیب تنظیم شود.
مقدار age و city در ایندکس دوم باید به ترتیب برابر با 23 و اصفهان باشد.
همچنین، برای ایندکس‌های زوج مقدار isActive برابر با false قرار گیرد.

بخش دوم: فیلتر و بازسازی داده

آرایه ساخته‌شده را پیمایش کنید و یک خروجی مطابق با ساختار زیر بسازید:
یک آبجکت با دو کلید even و odd، که هر کدام شامل یک آبجکت parent هستند. داخل parent اطلاعات مشخص شده ذخیره شود، و یک کلید children که شامل لیستی از آبجکت‌هایی است که id آن‌ها به ترتیب زوج یا فرد است. 

{
  "even": {
    "parent": {
      "id": 1,
      "name": "Alireza",
      "email": "alireza@example.com",
      "age": 27,
      "city": "Tehran",
      "isActive": false,
      "role": "admin",
      "children": [ /* آبجکت‌هایی با آی دی زوج */ ]
    }
  },
  "odd": {
    "parent": {
      "id": 2,
      "name": "Alireza",
      "email": "alireza@example.com",
      "age": 23,
      "city": "Isfahan",
      "isActive": true,
      "role": "admin",
      "children": [ /* آبجکت‌هایی با آی دی فرد */ ]
    }
  }
}
